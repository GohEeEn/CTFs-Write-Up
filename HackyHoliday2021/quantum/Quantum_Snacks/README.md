### Task 1

1. X(1, 0) = (0, 1)
2. Z(1, 0) = (1, 0)
3. Z(0, 1) = (0, -1)
4. H(1, 0) = sqrt(2) x (1, 1)
5. H(0, 1) = (1, -1)
6. Z x sqrt(2) x (1, 1) = sqrt(2) x (1, -1)
7. X x sqrt(2) x (1, -1) = sqrt(2) x (-1, 1)
8. Z x sqrt(2) x (-1, 1) = sqrt(2) x (-1, -1)

H x sqrt(2) x (1, 1) = (1, 0)
H x sqrt(2) x (1, -1) = (0, 1)
H x sqrt(2) x (-1, 1) = (0, -1)
H x sqrt(2) x (-1, -1) = (-1, 0)

-> __8__ different states
